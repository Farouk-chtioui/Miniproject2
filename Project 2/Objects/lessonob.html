How to Create Objects in JavaScript

In JavaScript, objects are used to store collections of data and functions. Objects can contain properties and methods, which are accessed using dot notation.

To create an object, you can use the {} notation, or the Object() constructor function. Here's an example:

let person = {
  name: "John",
  age: 30,
  hobbies: ["reading", "cooking", "running"],
  greet: function() {
    console.log("Hello, my name is " + this.name + ".");
  }
};

let car = new Object();
car.make = "Honda";
car.model = "Civic";
car.year = 2022;

The first object, person, has properties for name, age, and hobbies, as well as a method called greet. The second object, car, is created using the Object() constructor function and has properties for make, model, and year.

Example

Let's create an object called person with properties for name, age, and email, as well as a method called introduce.

let person = {
  name: "Alice",
  age: 25,
  email: "alice@example.com",
  introduce: function() {
    console.log("Hi, my name is " + this.name + ", and I'm " + this.age + " years old.");
  }
};

person.introduce(); // Hi, my name is Alice, and I'm 25 years old.

In this example, we create a new object with properties for name, age, and email, as well as a method called introduce. The method uses the this keyword to refer to the current object's properties.
