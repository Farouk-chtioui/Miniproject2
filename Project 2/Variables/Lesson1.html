How to Declare a Variable in JavaScript

Variables are used to store data in a program. In JavaScript, you can declare a variable using one of three keywords: var, let, or const.

var name = "John";
let age = 30;
const PI = 3.1416;

To declare a variable, use one of the keywords followed by the variable name and an optional initial value. The var keyword is used to declare a variable in the global scope or function scope. The let keyword is used to declare a block-scoped variable. The const keyword is used to declare a block-scoped variable that cannot be reassigned.

Example

Let's declare a variable called message and assign it a value of "Hello, World!".

let message = "Hello, World!";

Now, let's log the value of message to the console.

console.log(message);
