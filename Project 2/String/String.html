Strings in JavaScript

A string is a sequence of characters enclosed in quotation marks, either single quotes (') or double quotes ("). In JavaScript, you can use strings to store and manipulate text data.

let greeting = "Hello, world!";
let name = 'Alice';

To declare a string, use either single quotes or double quotes to enclose the text you want to store. You can then assign the string to a variable using the = operator.

Example

Let's declare a string called message and assign it a value of "Hello, JavaScript!".

let message = "Hello, JavaScript!";

We can then use various methods to manipulate the string. For example, we can use the toUpperCase() method to convert the string to uppercase:

console.log(message.toUpperCase());
          HELLO, JAVASCRIPT!

We can also use the charAt() method to get the character at a specific position in the string:

console.log(message.charAt(0));
          H
