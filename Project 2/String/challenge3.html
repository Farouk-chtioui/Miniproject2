Manipulate Strings

Write a JavaScript code snippet that performs the following tasks:

1. Create a string variable 'greeting' with the value "Hello, world!".

2. Print the length of the 'greeting' string to the console.

3. Convert the 'greeting' string to uppercase and store it in a new variable 'greetingUpperCase'.

4. Print the 'greetingUpperCase' string to the console.

5. Extract the word 'world' from the 'greeting' string and store it in a new variable 'world'.

6. Print the 'world' variable to the console.
