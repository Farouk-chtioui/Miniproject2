Operators Challenge

Write a Function

Write a JavaScript function called calculateGrade that takes a number as input and returns a string indicating the corresponding letter grade based on the following grading scale:

  • 90 or above: "A"
  • 80-89: "B"
  • 70-79: "C"
  • 60-69: "D"
  • Below 60: "F"

The input number will always be between 0 and 100.
