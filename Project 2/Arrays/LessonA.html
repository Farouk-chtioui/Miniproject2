How to Declare an Array in JavaScript

Arrays are used to store multiple values in a single variable. In JavaScript, you can declare an array using square brackets ([]) and separate each value with a comma.

let fruits = ["apple", "banana", "orange"];

To declare an array, use square brackets and list the values you want to store, separated by commas. You can also declare an empty array by using square brackets with no values inside.

Example

Let's declare an array called numbers and assign it some values.

let numbers = [1, 2, 3, 4, 5];

Now, let's log the first element of numbers to the console.

console.log(numbers[0]);
