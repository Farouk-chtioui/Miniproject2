How to Declare a Boolean in JavaScript

Booleans are a fundamental data type in JavaScript. They are used to represent true or false values in a program. A boolean expression is an expression that evaluates to either true or false. Booleans are commonly used in conditional statements, loops, and functions.

In JavaScript, you can declare a boolean using the true or false keyword. When a boolean value is declared, it is stored in the computer's memory as either a 1 or a 0.

It is also possible to create boolean expressions by comparing values. For example, you can use the comparison operators ==, !=, >, <, >=, and <= to compare values and create boolean expressions. The result of a comparison is always a boolean value.

let isGood = true;
let isBad = false;

To declare a boolean, use one of the keywords followed by the boolean name and an optional initial value.

Example

Let's declare a boolean called isStudent and assign it a value of true.

let isStudent = true;

Now, let's log the value of isStudent to the console.

console.log(isStudent);
