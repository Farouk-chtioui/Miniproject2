How to Define a Function in JavaScript

A function is a block of code that can be called to perform a specific task. In JavaScript, you can define a function using the function keyword.

function addNumbers(a, b) {
  return a + b;
}

To define a function, use the function keyword followed by the function name, a list of parameters enclosed in parentheses, and the function code enclosed in curly braces.

The return keyword is used to specify the value that the function should return when it is called.

Example

Let's define a function called multiplyNumbers that takes two parameters and returns their product.

function multiplyNumbers(a, b) {
  return a * b;
}

Now, let's call the multiplyNumbers function with arguments 2 and 3.

console.log(multiplyNumbers(2, 3));
          6

The function should return 6, which is the product of 2 and 3.
